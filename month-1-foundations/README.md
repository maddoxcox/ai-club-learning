# Month 1: Foundations — Python + JavaScript + APIs

**20 projects. 4 weeks. Mar 17 - Apr 11.**
**Schedule:** 1-2 hours/day, Mon-Fri mornings (8-10am), weekends off.

## How to Use

1. Navigate to today's project folder
2. Read `instructions.md`
3. Open the starter file
4. Build it with Claude Code helping you
5. Push to GitHub when done

## By the End of Month 1, You Can:
- Handle errors gracefully in Python (try/except, custom exceptions)
- Use pip to install and work with third-party packages
- Write tests with pytest
- Read/write CSV and JSON files
- Use regex for pattern matching and data cleaning
- Build with classes, inheritance, and OOP patterns
- Manipulate the DOM with vanilla JavaScript
- Call REST APIs from both JS (fetch) and Python (requests)
- Explain HTTP methods, status codes, JSON, and async/await
