# Day 14: Interactive Todo List

## What You're Building
A web-based todo list using vanilla JavaScript and DOM manipulation. No frameworks — pure JS.

## What You'll Learn
- `document.querySelector()` / `querySelectorAll()`
- `createElement()` / `appendChild()`
- `addEventListener()` — click, keypress, change events
- Event delegation — handling events on dynamic elements
- `localStorage` — persisting data in the browser

## Requirements
1. **Add todos** — text input + Add button (also Enter key)
2. **Display todos** — checkbox, text, delete button
3. **Complete todos** — checkbox toggles strikethrough
4. **Delete todos** — X button removes item
5. **Filter** — All / Active / Completed buttons
6. **Counter** — "3 items left"
7. **Clear completed** — remove all done items
8. **Persistence** — localStorage saves between page refreshes

## How to Test
Open `index.html` in your browser (double-click or `open index.html`).

## Stretch Goals
- Drag-and-drop reordering
- Due dates
- Edit on double-click
- Keyboard shortcuts
